# MLBB Draft Strategizer

MLBB Draft Strategizer is a web app concept for Mobile Legends: Bang Bang players who want a cleaner way to plan drafts before ranked or competitive matches.

The project is focused on helping players think through picks, bans, counters, team composition, and rotations in one place. Instead of only choosing heroes one by one, the app should act like a strategy board where players can build a draft, review possible counter picks, spot team weaknesses, and map out early-to-late game plans.

## What This Project Is About

The app will be designed around the MLBB draft experience:

- Planning blue team and red team picks and bans.
- Reviewing hero counters and matchup ideas.
- Checking if a team composition has enough frontline, damage, control, and scaling.
- Creating simple rotation plans on an MLBB map.
- Saving draft strategies for later review.

## Project Status

This repository is currently in the planning stage. The main goal right now is to define the project clearly before building the first version.

The first version should focus on making the core draft planning experience useful, smooth, and easy to understand.
